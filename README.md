# Hobby Connection

## サイト概要
　共通の趣味や仕事で個々でチャットを介してやり取りをすることができるアプリケーション。

### サイトテーマ
人とのつながりや出会いを増やす

### テーマを選んだ理由
普段の生活ではかかわることのない人とつながることができることによって、使用する人にとって新しいことを学べたりして人生の経験値になると思ったから。

### ターゲットユーザ
20代後半までの若いユーザー

### 主な利用シーン
興味のあること、趣味の仲間を作りたいとき。
